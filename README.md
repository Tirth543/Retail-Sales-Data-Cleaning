# Retail Store Sales Data Cleaning

This project focuses on cleaning and preparing a retail store sales dataset for analysis.
The original data contained missing values, incorrect data types, and duplicate records, which were handled using Python and Pandas.

## Tools Used
- Python
- Pandas
- Google Colab

## Files
- Retail_Store_Sales_Cleaning.ipynb – Data cleaning steps and explanations  
- retail_store_sales_cleaned.csv – Final cleaned dataset

The cleaned dataset is structured and ready for further analysis or visualization.
